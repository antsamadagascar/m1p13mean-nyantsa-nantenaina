## Prérequis ( Linux ubuntu 24.4)

- Node.js v20+
- Angular CLI v17+
- MongoDB v8+

---

## Installation rapide

### 1. Cloner le projet

```bash
git clone https://github.com/antsamadagascar/m1p13mean-nyantsa-nantenaina.git
cd m1p13mean-nyantsa-nantenaina
```

### 2. Backend

```bash
cd backend
npm install
npm run dev
```

### 3. Frontend

```bash
cd frontend
npm install
ng serve --open
```

### 4. MongoDB

```bash
# Lancer MongoDB
mongod --dbpath /var/lib/mongodb
```
